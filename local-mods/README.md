# Supplied local runtime mods

These JARs were supplied by the project owner for local development. `.gitignore` excludes JAR files: do not publish them to GitHub without confirming each author's redistribution permission.

- `campchat-1.0.0.jar`: Forge 47+, Minecraft 1.20.1, Java 17, All Rights Reserved, author `kltyton`. This is the CampChat version used by the port; `campchat-fabric` is retired.
- `stalker_decor_mod_1.20.1.jar`: Forge 47+, exact Minecraft 1.20.1, Java 17, license not specified.
- `stalkercubedreborn_1.0.9_fix_forge.jar`: Forge 47+, exact Minecraft 1.20.1, Java 17, requires Curios 5.14+ and GeckoLib 4.8.3+.

CampChat directly references `xaero.map.gui.GuiMap`, `xaero.map.WorldMapSession`, and `xaero.map.MapProcessor`. Install a compatible Xaero's World Map Forge 1.20.1 JAR on the client.
